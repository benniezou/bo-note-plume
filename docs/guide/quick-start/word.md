掌握 Word 的基本操作是高效办公的基础。无论你是正在准备计算机等级考试，还是为了应对日常工作需求，熟悉这些核心功能都能事半功倍。
我为你梳理了 Word 最常用、最核心的基本操作，涵盖了从文档创建到排版、编辑的全流程。
1. 文档的创建与保存 📂
这是使用 Word 的第一步，确保你的劳动成果不会丢失。
新建文档： 启动 Word 时通常会看到模板库，你可以选择“空白文档”开始，也可以使用简历、报告等模板。
保存文档：
首次保存： 按 Ctrl + S 或点击快速访问工具栏的软盘图标，选择保存位置并命名。
另存为： 如果想保留原文件不变，可以使用“文件”>“另存为”，修改文件名或路径进行保存。
自动恢复： 如果遇到断电等意外，重新打开 Word 时通常会在左侧看到“文档恢复”面板，可以找回未保存的内容。
2. 文本的输入与编辑 ✍️
这是最频繁的操作，包括输入内容和修改内容。
输入与光标： 闪动的竖线是“插入点”，所有输入的内容都会出现在这里。你可以通过鼠标点击或方向键移动光标。
插入与改写模式：
插入模式（默认）： 输入文字时，光标后的文字会自动后退。
改写模式： 输入文字时，光标后的文字会被新输入的文字覆盖。你可以在状态栏查看或切换该模式（通常双击状态栏的“改写”或按 Insert 键切换）。
基础编辑快捷键：
撤销/恢复： Ctrl + Z（撤销误操作），Ctrl + Y（恢复被撤销的操作）。
复制/剪切/粘贴： Ctrl + C（复制），Ctrl + X（剪切），Ctrl + V（粘贴）。如果需要保留原格式或只粘贴文本，可以使用“选择性粘贴”。
特殊字符输入： 按 Ctrl + Shift + 8 可以显示/隐藏段落标记（如空格、回车符）。输入注册商标™可用 Ctrl + Alt + T，版权©可用 Ctrl + Alt + R。
3. 格式设置：字符与段落 🎨
让文档从“能看”变成“好看”。
字符格式： 在“开始”选项卡中，可以设置字体、字号、加粗（Ctrl + B）、倾斜（Ctrl + I）、下划线（Ctrl + U）以及字符颜色。
段落格式： 选中文本后，在“段落”组中可以设置：
对齐方式： 左对齐、居中、右对齐、两端对齐（Ctrl + J）。
缩进与间距： 首行缩进（通常中文段落首行缩进 2 字符）、行距（如 1.5 倍行距）。
格式刷： 双击“格式刷”按钮，可以连续多次应用同一格式，非常高效。
4. 查找、替换与审阅 🔍
处理长文档时的神器。
查找与替换： 按 Ctrl + F 查找，Ctrl + H 替换。这不仅限于文字，还可以替换字体格式或进行高级的通配符查找。
拼写检查： 文字下方的红色波浪线表示拼写可能有误，绿色波浪线表示语法可能有误。
批注与修订：
批注： 在侧边栏添加评论，不影响正文内容。
修订： 开启“修订”模式后，所有的增删改操作都会被标记出来（通常是不同颜色的下划线或删除线），方便多人协作审阅。
5. 页面设置与打印 🖨️
最后一步，让文档符合打印或提交的标准。
页面布局： 设置页边距、纸张方向（纵向/横向）、纸张大小（如 A4）。
页眉页脚： 双击页面顶部或底部区域，可以插入公司 Logo、页码或日期。通常页码放在页脚。
分页与分节： 按 Ctrl + Enter 可以快速插入分页符。如果文档前后部分需要不同的页眉页脚或页码格式，需要使用“分节符”。
