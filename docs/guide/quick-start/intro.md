## 登录管理
::: steps
1. 右键点击桌面空白处，点击`新建`、再点击`快捷方式`。
![img_1.png](img_1.png)

2. 在如下页面在输入`\\192.168.0.1` 并点击下一步。
![img.png](img.png)

3. 点击完成 桌面出现下图快捷方式，并打开。
![img_2.png](img_2.png)

4. 输入帐号和密码，如果无法登录，需要在账号前面填写 ==`gauchen0\账号`==

![img_3.png](img_3.png)
:::
::: danger
   密码规范：`字母`、`数字`、`符号` 必须包含两种.!!!

:::
