## 🧮 基础统计函数
这是 Excel 的基石，用于最基础的数据汇总和分析。
### SUM (求和)
功能：计算指定区域内所有数值的总和。
语法：=SUM(数值区域)
案例：=SUM(A1:A10) 会计算 A1 到 A10 单元格中所有数字的总和。
### AVERAGE (平均值)
功能：计算指定区域内数值的算术平均值，会自动忽略空单元格和文本。
语法：=AVERAGE(数值区域)
案例：=AVERAGE(B2:B20) 用于计算 B2 到 B20 单元格区域的平均分数。
### MAX / MIN (最大值/最小值)
功能：找出一组数据中的最大值或最小值。
语法：=MAX(数值区域) 或 =MIN(数值区域)
案例：=MAX(C1:C100) 可以找出 C 列中的最高销售额。
## ✅ 逻辑判断函数
用于根据条件返回不同的结果，是实现数据自动分类的关键。
### IF (条件判断)
功能：根据设定的条件判断真假，并返回对应的结果。
语法：=IF(条件, 条件为真时的结果, 条件为假时的结果)
案例：=IF(D2>=60, "及格", "不及格")，如果 D2 单元格的分数大于等于 60，就显示“及格”，否则显示“不及格”。
### IFS (多条件判断)
功能：同时判断多个条件，适用于 Excel 2016 及更新版本。它会从左到右依次检查条件，并返回第一个为真的条件对应的结果。
语法：=IFS(条件1, 结果1, 条件2, 结果2, ...)
案例：=IFS(E2>=90, "优秀", E2>=80, "良好", E2>=60, "及格", TRUE, "不及格")，根据分数区间自动评定等级。
## 🔍 查找与引用函数
在海量数据中快速定位并提取所需信息。
VLOOKUP (垂直查找)
功能：在表格或区域的第一列中查找指定的值，并返回该行中指定列的数据。
语法：=VLOOKUP(查找值, 查找区域, 返回值在区域中的第几列, 精确匹配)
案例：=VLOOKUP("张三", A2:D10, 3, FALSE)，在 A2 到 D10 的区域中查找“张三”，并返回该行第 3 列（比如“工资”列）的值。注意：第四个参数通常设为 FALSE 以进行精确匹配。
## 📊 条件统计函数
根据特定条件对数据进行求和或计数。
### COUNTIF (条件计数)
功能：统计满足指定条件的单元格数量。
语法：=COUNTIF(条件区域, 条件)
案例：=COUNTIF(B1:B100, ">100")，统计 B 列中数值大于 100 的单元格个数。
### SUMIF (条件求和)
功能：对满足特定条件的单元格进行求和。
语法：=SUMIF(条件区域, 条件, 求和区域)
案例：=SUMIF(F2:F10, "销售部", G2:G10)，汇总“销售部”在 G 列对应的所有销售额。
### COUNTIFS / SUMIFS (多条件统计)
功能：COUNTIFS 用于统计满足多个条件的单元格数量，SUMIFS 用于对满足多个条件的单元格进行求和。
语法：=COUNTIFS(条件区域1, 条件1, 条件区域2, 条件2, ...) 或 =SUMIFS(求和区域, 条件区域1, 条件1, 条件区域2, 条件2, ...)
案例：=SUMIFS(G2:G10, F2:F10, "销售部", H2:H10, ">5000")，统计“销售部”且“销售额大于5000”的总和。
## ✂️ 文本处理函数
用于清洗和整理不规范的文本数据。
### LEFT / RIGHT / MID (截取文本)
功能：分别用于从文本的左侧、右侧或中间位置提取指定数量的字符。
语法：
=LEFT(文本, 字符数)
=RIGHT(文本, 字符数)
=MID(文本, 开始位置, 字符数)
案例：=RIGHT(A1, 3) 会从 A1 单元格的文本末尾提取最后 3 个字符。
::: info
所有函数都必须以等号 = 开头。你可以直接在单元格中输入，也可以使用“公式”选项卡中的“插入函数”向导来辅助输入。
:::
